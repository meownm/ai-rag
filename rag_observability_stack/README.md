# RAG Observability Stack v5 (Windows + Docker Desktop)

Эта версия добавляет автообнаружение FastAPI-сервисов (Prometheus Docker SD).
LLM-инстансы автообнаружением не покрываются.

Стек: Phoenix, Helicone, Prometheus, Grafana, Traceloop, Nginx proxy (BasicAuth).
Поддержка: multi-service, multi-model (в метриках), две роли PostgreSQL.
