# Проверка на незакрытые конфликты

Добавлен скрипт `tools/check_conflicts.py`, который просматривает все отслеживаемые файлы и ищет маркеры незавершённых merge-конфликтов (`<<<<<<<`, `=======`, `>>>>>>>`).

## Как запустить
```bash
python tools/check_conflicts.py
```
Если маркеров нет, выводится сообщение:
```
No conflict markers detected in tracked files.
```
При наличии конфликтов скрипт завершится с кодом 1 и перечислит проблемные файлы.

## Зачем
Позволяет оперативно убедиться, что все конфликты действительно решены перед коммитом и PR, особенно после крупных слияний.
